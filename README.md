## What's this ?

This bot allows you to automatically delete all your messages, including DM.

## How do I use it ?

1. Insatll NodeJS

2. Edit the token in `index.js`, at the `client.login("")` field

```js
const Discord = require('discord.js')
const client = new Discord.Client();

const fs = require('fs');

client.login("")
```

Enjoy :)
